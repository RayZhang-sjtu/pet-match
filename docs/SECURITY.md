# 安全与隐私

- 不收集、上传或持久化答案，不使用统计、广告、账户或第三方脚本。
- 前端变量会暴露；任何密钥不得放入前端或仓库。
- 内容以文本渲染，禁止未经净化的 HTML 注入。
- 锁定依赖；新增依赖需说明用途、替代、维护和安全影响。
- 托管时启用 HTTPS、CSP、`X-Content-Type-Options`、`Referrer-Policy`、`Permissions-Policy` 和防非预期嵌入策略。
- 面向未成年人；未来加入分析、营销或个人数据前，必须评估目标地区法律、年龄门槛、监护人同意与隐私政策。

密钥泄露时需立即撤销/轮换并检查历史与日志；只删当前文件不足以消除泄露。
