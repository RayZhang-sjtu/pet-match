# 如何查看访问情况

当前网页不安装统计脚本、不设置分析 Cookie，也不把用户行为发送给第三方。

项目所有者可以打开 [GitHub Traffic](https://github.com/RayZhang-sjtu/pet-match/graphs/traffic)，查看最近 14 天的仓库浏览次数、独立访客、来源与热门内容。路径是仓库首页 → `Insights` → `Traffic`。数据约每小时或每天更新。

## 重要限制

GitHub 官方把这些数据定义为“仓库流量”，不是 GitHub Pages 网站的精确按钮点击统计。因此它适合确认项目是否有人访问、观察大致趋势，但不能准确回答“网页被打开多少次”或“开始测试按钮被点击多少次”。

若未来必须获得精确网页浏览或按钮点击数，需要接入分析服务或自建计数接口。这会产生网络请求并涉及访问数据、隐私说明和未成年人评估，必须由项目所有者另行确认后才能实施。
