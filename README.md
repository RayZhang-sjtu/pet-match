# Pet Match 宠物人格测试

一个面向 10–30 岁用户的中文互动网页：通过 30 个生活方式问题，给出宠物类型与代表性品种/类型的初步匹配建议。

当前为 MVP 内容草案 v0.1。它不是心理测评，也不能代替对住房、预算、过敏、当地法规、长期照护和具体动物个体的真实评估。答案仅在当前页面内存中计算，不上传、不保存。

## 当前状态

- 已完成：30 题完整流程、透明本地评分、5 类结果、现实约束提示、响应式页面、基础自动检查。
- 已确认：Pet Match 品牌、中国大陆简体中文用户、绿粉圆角风格、明确品种结果、结果分享。
- 待处理：专业内容复核、交互人工验收、临时托管平台。
- 不包含：账户、数据库、统计追踪、支付、后台、答案和进度持久化。

不熟悉这些网站概念时，请先阅读 [网站决策通俗说明](docs/WEBSITE_DECISION_GUIDE.md)。

## 本地运行

需要 Node.js 22.13 或更高版本。运行 `npm ci`，再运行 `npm run dev`。提交修改前运行 `npm run check`。

## 非技术验收

打开页面后开始测试；确认未选择答案时不能下一题；完成 30 题；确认结果包含类型、示例、原因、现实核对项与免责声明；点击“重新测试”后回到首页。
